How to use the ODDESSA search tool

The oddessa search engine allows you to look for data elements across all mapped national collections and datsets.

To search for a particular term just start typing in the search box. ODDESSA will present a list of options based on what you type to make searches more specific, you are free to ignore these choices and enter the terms you want.

The radio buttons beneath the search text box specify the type of search 'Elements' searches for matches in all the known data elements ODDESSA contains, 'SIF' allows you to search for all elements that map to the SIF objects you choose, 'Collections' is simply a shorthand way to retrieve all elements of a collection at once.

The search accepts multiple terms separated by commas allowing you to make search requests for multiple elements, sif objects, or collections at the same time.

When you have as many terms as you want to search with, press the 'Go' button to perform the search.

Search results are returned below the search bar showing major features of each element and the collection they come from.

To see the extended meta-data for each item click on the 'Read More' button for that search result.

By selecting the 'pin' option on a search result it is maintained as part of the search results between searches. This allows you to construct new datasets based on elements of existing collections.

Use the 'Map' menu items to see the mapping of your search results to the SIF National Use Cases and the SIF-AU Data Model.
